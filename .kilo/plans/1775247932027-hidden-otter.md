## Remediation Plan

### Phase 1: Dependency Upgrades

**1.1 Upgrade Next.js to 16.1.7**
```bash
npm install next@16.1.7
```
- Resolves 8 Dependabot alerts (#8-15)
- Run `npm run build` to verify compatibility
- Run tests (`npm run test`) to verify

---

### Phase 2: Code Scanning Fixes

**2.1 Fix file system race conditions (scripts/fix-middleware-nft.js)**
- Lines 27, 49, 56, 64: `fs.existsSync()` before `fs.writeFileSync()`
- Fix: Use `fs.openSync()` with `fs.O_CREAT | fs.O_EXCL` flags for atomic creation
- Alternatively: Use `fs.writeFileSync()` directly (will fail if exists, safe)

**2.2 Add workflow permissions (hubbopos-sync.yml)**
- Add `permissions:` block with minimal required permissions
- `contents: read`, `pull-requests: write` (if needed)

**2.3 Fix log injection (lalamove/route.ts)**
- Lines 85, 115: User-provided values logged without sanitization
- Fix: Sanitize user input by removing newlines before logging
- Pattern: `value.replace(/\n|\r/g, "")`

**2.4 Pin action versions (dependabot-auto-merge.yml)**
- Pin `dependabot/fetch-metadata@v2` to commit SHA
